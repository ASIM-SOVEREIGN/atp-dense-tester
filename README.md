**Automated testing harness for the Agent Transaction Protocol (ATP)**

This agent generates signed intents, sends them to ATP-enabled sovereigns (VEXR Ultra, ASIM_Pilot, etc.), verifies cryptographic receipts, and builds comparative reports.
